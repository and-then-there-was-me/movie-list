# code-challenge-village-cinema

## Getting started

### Project setup

Make sure to use node v10.15.3 and npm v6.4.1 or above

Install project dependencies:

```
npm install
```

#### Run development server

```
npm start
```

#### Run linters

```
npm run lint
```

#### Create production build

```
npm run build
```

#### Serve dist directory

```
npm run serve-dist
```
